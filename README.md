# Active Directory Lab - Windows Server 2022

Laboratorio práctico de Active Directory realizado con Windows Server 2022 y máquinas virtuales Windows 10.

## Objetivo
Simular un entorno empresarial básico mediante la configuración de Active Directory Domain Services, autenticación de dominio y administración de usuarios.

## Tecnologías utilizadas
- Windows Server 2022
- Active Directory Domain Services (AD DS)
- Windows 10
- VirtualBox
- Networking
- DNS
- Autenticación de dominio

## Infraestructura

| Máquina | Función |
|---|---|
| DC01 | Controlador de Dominio |
| CLIENT0 | Cliente unido al dominio |

## Funcionalidades implementadas
- Configuración de IP estática
- Instalación de Active Directory
- Promoción a Domain Controller
- Configuración de dominio
- Creación y administración de usuarios
- Unión de cliente Windows al dominio
- Autenticación mediante dominio
- Pruebas de conectividad
